:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
提供儲存與運算資源、結合這些資源、打造出以VMware軟體功能為後盾的VMware vSphere環境。NetApp HCI NetApp Element

您可以使用NetApp混合雲控制介面來升級、擴充及監控系統、NetApp HCI 並使用NetApp Element 適用於vCenter Server的VMware外掛程式來管理各種VMware資源。
