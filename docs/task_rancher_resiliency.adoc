= 設定恢復能力
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
瞭解如何在NetApp HCI 將Rancher安裝到位後、設定虛擬機器放置位置以獲得最佳恢復能力。

您可能會考慮在下列方面恢復能力：

* 如果您設定負載平衡或循環配置資源Proxy、管理叢集就會恢復彈性。
* 對於使用者叢集、您可以根據要設定這些使用者叢集的方式來設定恢復能力。
