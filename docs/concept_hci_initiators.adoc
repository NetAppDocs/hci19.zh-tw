= 啟動器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
啟動器可讓外部用戶端存取叢集中的磁碟區、做為用戶端與磁碟區之間通訊的入口點。您可以使用啟動器進行CHAP型存取、而非帳戶型存取儲存磁碟區。新增至Volume存取群組的單一啟動器、可讓Volume存取群組成員存取新增至群組的所有儲存磁碟區、而無需驗證。啟動器只能屬於一個存取群組。
